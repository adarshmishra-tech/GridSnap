# GridSnap

GridSnap is a block-based puzzle game built with React Native (Expo).  
The game focuses on simple mechanics, smooth animations, and continuous gameplay that gradually gets harder over time.

---

## Features

- Block placement-based gameplay
- Score and combo system
- Particle effects for clears and actions
- Sound effects for game events
- Power-ups to change gameplay flow
- Endless progression with increasing difficulty
- Mobile-first design (Android support)

---

## How the game works

You place different shaped blocks on a grid.  
When a row or valid pattern is completed, it gets cleared and you earn points.

As you keep playing:
- The pace increases
- The board becomes more challenging
- Combos give higher scores

There is no fixed end — the game continues until no more moves are possible.
